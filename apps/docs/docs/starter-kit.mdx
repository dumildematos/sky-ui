## Live Playground

```jsx live
function StarterKitLive() {
  const [theme, setTheme] = React.useState('midnight');

  return (
    <SkyShell theme={theme} fullHeight={false}>
      <SkyCard className="p-6">
        <div className="mb-4 flex flex-wrap gap-3">
          <SkyButton color="cyan" theme={theme} onClick={() => setTheme('midnight')}>Midnight</SkyButton>
          <SkyButton color="amber" theme={theme} onClick={() => setTheme('daylight')}>Daylight</SkyButton>
        </div>
        <div className="rounded-2xl border border-white/8 bg-white/5 p-4">
          <div className="text-sm uppercase tracking-[0.28em] text-slate-400">Preview Surface</div>
          <div className="mt-3 text-2xl font-light">SkyCard + SkyButton + SkyShell</div>
        </div>
      </SkyCard>
    </SkyShell>
  );
}
```
